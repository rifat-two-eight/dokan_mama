২৫. Order History Module
Overview

Order History Module-এর মাধ্যমে ব্যবহারকারী তার সকল অর্ডার দেখতে, সার্চ করতে, ফিল্টার করতে, ট্র্যাক করতে এবং প্রতিটি অর্ডারের বিস্তারিত তথ্য দেখতে পারবে। এছাড়াও নির্দিষ্ট শর্ত পূরণ হলে Delivery Method পরিবর্তন, Change Request এবং Order Tracking-এর মতো ফিচার ব্যবহার করতে পারবে।

এই মডিউল মূলত দুইটি পেজ নিয়ে গঠিত:

Order History Page (Order List)
Order Details Page (View Order)

UX Guidelines
Order Timeline সবসময় পেজের উপরে দৃশ্যমান থাকবে।
বর্তমান (Current) Status স্পষ্টভাবে Highlight করা হবে।
প্রতিটি Timeline Step-এর সাথে Date এবং Time প্রদর্শন করা হবে।
Order Status অনুযায়ী Action Button স্বয়ংক্রিয়ভাবে পরিবর্তিত হবে।
যেসব Action আর করা সম্ভব নয়, সেগুলো Disabled অবস্থায় থাকবে এবং কারণ দেখানো হবে।
Track Order নির্বাচন করলে প্রয়োজনে একটি Live Map বা Delivery Tracking Screen-এ নিয়ে যাওয়া যেতে পারে।
পুরো Order History Module Mobile এবং Desktop উভয় প্ল্যাটফর্মের জন্য Responsive এবং সহজে ব্যবহারযোগ্য (User-Friendly) হতে হবে।
